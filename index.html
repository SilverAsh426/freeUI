简洁至上

Free UI 是一个轻便简洁的 UI 框架，提供了 button,grid,input,layout 等常用组件，同时满足移动端和 PC 端的适配。

Vue驱动

享受 Vue + webpack 的开发体验，在 Markdown 中使用 Vue 组件，同时可以使用 Vue 来开发自定义主题。

持续更新

目前框架库中为部分组件，后续将不断加入新组件。
